# Employee HR Analytics Project

## Project Overview
This project was built using MySQL Workbench to analyze employee data and HR analytics.

## Features
- Employee database management
- Salary analysis
- Attrition analysis
- Department performance analysis
- Stored procedures
- SQL triggers
- HR reporting

## Technologies Used
- MySQL Workbench
- SQL

## Dataset
IBM HR Analytics Employee Attrition Dataset

## SQL Concepts Used
- SELECT
- GROUP BY
- ORDER BY
- Aggregate Functions
- Views
- Stored Procedures
- Triggers

## Sample Queries

```sql
SELECT Department,
AVG(MonthlyIncome)
FROM employees
GROUP BY Department;
```
